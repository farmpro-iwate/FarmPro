# FarmPro Starter Pack 50 月別収支 CSV出力・印刷追加版

## 内容

Starter Pack 50では、月別収支画面にCSV出力と印刷機能を追加します。

- 月別収支画面に「印刷」ボタンを追加
- 月別収支表をCSVで保存
- 月別収支表を紙やPDFで保存
- 印刷日時を表示
- 印刷時はボタン類を非表示
