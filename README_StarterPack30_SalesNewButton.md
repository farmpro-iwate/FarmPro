# FarmPro Starter Pack 30 出荷・販売一覧に新規登録ボタン追加版

## 内容

Starter Pack 30では、出荷・販売一覧画面に「新規登録」ボタンを追加します。

## 更新ファイル

- client/src/pages/SalesList.tsx
- docs/StarterPack30_確認手順.md

## 追加されること

出荷・販売管理画面の上部に「新規登録」ボタンが出ます。
クリックすると、次の画面へ移動します。

```text
/sales/new
```
