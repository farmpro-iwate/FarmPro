# 繁殖Farm Pro - Starter Pack 21 ヘルプ・操作ガイド

Starter Pack 21 では、アプリ内に「ヘルプ・操作ガイド」画面を追加します。

## 追加内容

- 上部メニューに「ヘルプ」追加
- ホーム画面の作業ショートカットに「ヘルプ」追加
- 初めて使う人向けの操作ガイド
- 毎日の作業の流れ
- 各機能の説明
- データ保存・バックアップの説明
- よくあるトラブル対策
- 印刷ボタン

## 便利になること

機能が増えてきたので、使い方をアプリ内で確認できるようにします。
農場スタッフや家族に使ってもらう時にも説明しやすくなります。
