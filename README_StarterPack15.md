# 繁殖Farm Pro - Starter Pack 15 牛選択・入力補助

Starter Pack 15 では、登録済みの牛を選んで入力できる補助機能を追加します。

## 追加内容

- 牛選択コンポーネント
- 登録済み牛から選択
- 個体番号・牛名を自動入力
- 手入力も継続可能
- 対応画面
  - 繁殖登録
  - BLV登録
  - ワクチン登録
  - 予定登録
  - 治療登録

## 便利になること

これまで毎回、個体番号と牛名を手入力していました。
Starter Pack 15 からは、登録済み牛を選ぶだけで自動入力できます。
