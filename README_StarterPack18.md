# 繁殖Farm Pro - Starter Pack 18 子牛カルテ

Starter Pack 18 では、子牛ごとの「子牛カルテ」画面を追加します。

## 追加内容

- 子牛管理から子牛カルテを開く
- 子牛の基本情報を確認
- ワクチン記録の確認
- 予定の確認
- 治療記録の確認
- 印刷ボタン
- 子牛ごとの履歴確認がしやすくなる

## 便利になること

Starter Pack 17 では成牛の個体カルテを追加しました。
Starter Pack 18 では子牛のカルテを追加します。

子牛ごとに、ワクチン・予定・治療の履歴をまとめて確認できます。
