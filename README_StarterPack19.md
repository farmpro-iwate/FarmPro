# 繁殖Farm Pro - Starter Pack 19 月間カレンダー

Starter Pack 19 では、予定を月単位で確認できる「月間カレンダー」を追加します。

## 追加内容

- 月間カレンダー画面
- 前月・翌月の切り替え
- 予定管理の予定を表示
- 繁殖記録の分娩予定日を表示
- ワクチンの次回予定日を表示
- BLVの次回検査予定日を表示
- 今日の日付を強調表示
- 印刷ボタン
- ホーム画面の作業ショートカットにカレンダー追加
- 上部メニューにカレンダー追加

## 便利になること

予定管理だけでなく、分娩予定・ワクチン予定・BLV検査予定も月間カレンダーでまとめて見られます。
