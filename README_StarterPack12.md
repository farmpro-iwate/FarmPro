# 繁殖Farm Pro - Starter Pack 12 検索・絞り込み強化

Starter Pack 12 では、各一覧画面に検索・絞り込みを追加します。

## 追加内容

- 牛台帳：個体番号・名号検索、BLV状態フィルター
- 子牛管理：子牛番号・名号・母牛名検索、性別フィルター
- 繁殖管理：個体番号・牛名・種雄牛検索、妊娠結果フィルター
- ワクチン管理：対象番号・対象名・ワクチン名検索、状態フィルター
- BLV管理：個体番号・牛名検索、検査結果フィルター
- 予定管理：タイトル・対象名・対象番号検索、予定区分・状態フィルター
- 治療管理：対象番号・対象名・症状・薬剤検索、経過フィルター
- 検索条件クリアボタン
