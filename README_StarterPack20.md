# 繁殖Farm Pro - Starter Pack 20 通知・期限アラート強化

Starter Pack 20 では、期限が近い作業や期限切れ作業をまとめて確認できる「アラート」画面を追加します。

## 追加内容

- アラート画面
- 上部メニューに「アラート」追加
- ホーム画面のショートカットに「アラート」追加
- 未完了予定の期限切れ・近日予定を表示
- 分娩予定の近日アラートを表示
- ワクチン予定の期限切れ・近日予定を表示
- BLV次回検査の期限切れ・近日予定を表示
- 治療中・要再診を表示
- 休薬期間中の治療記録を表示
- アラート件数を集計表示
- 印刷ボタン

## 便利になること

カレンダーでは月単位で予定を見られます。
Starter Pack 20 のアラートでは「今すぐ注意すべきもの」を一覧で確認できます。
