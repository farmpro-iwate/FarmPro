# 月別収支管理 設計メモ

## 1. 目的

出荷・販売の売上と、経費管理の支出を月別にまとめ、農場の収支を見える化する。

## 2. 使うデータ

### 売上

出荷・販売管理のデータを使う。

対象:

```text
status = 販売済み
saleDate が入力されているもの
salePrice が入力されているもの
```

### 経費

経費管理のデータを使う。

対象:

```text
paymentDate が入力されているもの
amount が入力されているもの
```

## 3. 月別収支で表示する項目

| 項目 | 内容 |
|---|---|
| 年月 | 2026-07 など |
| 売上合計 | 販売済みの販売金額合計 |
| 経費合計 | 経費の金額合計 |
| 差引収支 | 売上合計 - 経費合計 |
| 販売頭数 | 販売済み件数 |
| 平均販売金額 | 売上合計 ÷ 販売頭数 |
| 飼料費 | 経費区分が飼料費の合計 |
| 診療・医薬品 | 診療費と医薬品費の合計 |
| 繁殖費 | 種付け・繁殖費の合計 |
| その他経費 | その他の経費合計 |

## 4. 年別収支で表示する項目

- 年間売上
- 年間経費
- 年間収支
- 年間販売頭数
- 平均販売金額
- 月別一覧
- 経費区分別合計

## 5. 画面案

将来的に以下の画面を追加する。

```text
経営管理
  ├ 月別収支
  ├ 年別収支
  ├ 経費一覧
  └ 経費区分別集計
```

## 6. レポート連携案

レポート画面には、以下を追加する。

- 今月売上
- 今月経費
- 今月収支
- 今年売上
- 今年経費
- 今年収支

## 7. 注意点

この機能は経営判断用の目安として作る。

税務申告用の正式な帳簿・決算書は、会計ソフトや税理士との確認が必要。
