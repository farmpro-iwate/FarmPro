# 飼料在庫管理 設計メモ

## 1. 目的

飼料の入庫・出庫・残量を管理し、在庫不足や使用量を把握する。

## 2. 管理する内容

- 飼料名
- 入庫
- 出庫
- 調整
- 数量
- 単位
- 単価
- 金額
- 仕入先
- メモ

## 3. 在庫計算

在庫は以下で計算する。

```text
現在在庫 = 入庫合計 - 出庫合計 + 調整
```

## 4. 入庫の例

```text
日付: 2026-07-03
飼料名: 配合飼料
区分: 入庫
数量: 500
単位: kg
単価: 80
仕入先: JA
```

## 5. 出庫の例

```text
日付: 2026-07-04
飼料名: 配合飼料
区分: 出庫
数量: 120
単位: kg
メモ: 母牛群へ給与
```

## 6. 在庫不足候補

将来的に、在庫が少なくなった飼料をアラート表示する。

例:

```text
配合飼料の在庫が100kg以下
乾草ロールが3個以下
```

## 7. 経費管理との関係

飼料購入時の金額は経費管理の「飼料費」に登録する。

飼料在庫では数量管理を中心にする。

## 8. 開発順序

1. 飼料在庫 server API
2. 飼料在庫一覧
3. 入庫・出庫登録
4. 編集・削除
5. 在庫集計
6. CSV出力
7. 印刷
8. 在庫不足アラート
