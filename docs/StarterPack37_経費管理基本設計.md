# Starter Pack 37 経費管理 基本設計

## 1. 目的

繁殖和牛農家向けに、日々の支出を記録し、月別・年別の収支確認につなげる。

まずは難しい会計ソフトではなく、農場で実際に使いやすい「経費台帳」として作る。

## 2. 経費管理でできるようにすること

将来的に以下をアプリへ追加する。

- 経費一覧
- 経費新規登録
- 経費編集
- 経費削除
- 検索
- 経費区分で絞り込み
- 支払方法で絞り込み
- 月別集計
- 年別集計
- CSV出力
- 印刷
- レポート反映

## 3. 登録項目

経費記録では、以下の項目を登録する。

| 項目 | 内容 | 例 |
|---|---|---|
| 支払日 | 経費を支払った日 | 2026-07-03 |
| 経費区分 | 何の支出か | 飼料費 |
| 内容 | 詳細 | 配合飼料 |
| 支払先 | 店名・業者名 | JA、飼料会社 |
| 金額 | 支払金額 | 120000 |
| 支払方法 | 現金、口座、カードなど | 口座振替 |
| 対象 | 関連する牛・子牛・群など | 育成牛群 |
| メモ | 補足 | 7月分 |
| 作成日時 | 自動記録 |  |
| 更新日時 | 自動記録 |  |

## 4. 経費区分

最初は以下の区分を用意する。

- 飼料費
- 敷料費
- 医薬品費
- 診療費
- 種付け・繁殖費
- 購入牛費
- 水道光熱費
- 燃料費
- 修繕費
- 機械・資材費
- 車両費
- 保険料
- 手数料
- 消耗品費
- その他

## 5. 支払方法

最初は以下の支払方法を用意する。

- 現金
- 口座振替
- 銀行振込
- クレジットカード
- JA精算
- その他

## 6. 月別収支へのつなげ方

出荷・販売管理の売上と、経費管理の支出を月ごとに集計する。

月別収支では以下を出す。

- 売上合計
- 経費合計
- 差引収支
- 販売頭数
- 平均販売金額
- 主な経費区分別合計

## 7. レポート画面への追加案

レポート画面には、以下を追加する。

- 今月売上
- 今月経費
- 今月収支
- 今年売上
- 今年経費
- 今年収支
- 経費区分別合計
- 月別収支グラフ候補

## 8. CSV出力

経費管理では以下をCSV出力できるようにする。

- 経費一覧CSV
- 月別収支CSV
- 経費区分別CSV

## 9. 印刷

以下を印刷できるようにする。

- 経費台帳
- 月別収支表
- 年間収支表

## 10. 開発順序

安全に進めるため、次の順番で作る。

1. 経費管理 server API
2. 経費一覧画面
3. 経費新規登録
4. 経費編集
5. 経費削除
6. 検索・絞り込み
7. CSV出力
8. 印刷
9. レポート反映
10. 月別収支画面

## 11. 注意

本格的な税務申告や決算書作成は、税理士・会計ソフトとの連携を前提にする。

FarmProではまず、農場内で状況を把握しやすい経営管理を目指す。
