# 月別収支 画面設計メモ

## 1. 画面名

```text
月別収支
```

## 2. メニュー位置

将来的に上部メニューへ追加する。

おすすめ位置:

```text
出荷販売
経費管理
月別収支
レポート
```

## 3. 画面上部の集計カード

画面上部には以下を表示する。

- 売上合計
- 経費合計
- 差引収支
- 販売頭数
- 経費件数

## 4. 絞り込み

以下を追加する。

- 年
- 月
- 表示期間
- プラス収支のみ
- マイナス収支のみ

## 5. 一覧の列

| 列 | 内容 |
|---|---|
| 年月 | 2026-07 |
| 売上合計 | 販売済み合計 |
| 経費合計 | 経費合計 |
| 差引収支 | 売上 - 経費 |
| 販売頭数 | 販売済み件数 |
| 平均販売金額 | 売上 ÷ 販売頭数 |
| 平均販売体重 | 販売体重平均 |
| 経費件数 | 経費記録数 |
| 飼料費 | 飼料費合計 |
| 診療・医薬品 | 診療費 + 医薬品費 |
| 繁殖費 | 種付け・繁殖費 |
| その他経費 | その他経費 |

## 6. CSV出力

表示中の月別収支をCSV出力する。

ファイル名例:

```text
farmpro_monthly_balance_20260703.csv
```

## 7. 印刷

月別収支表として印刷できるようにする。

印刷時は検索条件やボタンを非表示にする。

## 8. 注意表示

画面に以下の注意を表示する。

```text
この収支は入力データをもとにした概算です。
正式な会計・税務申告には会計ソフトや税理士の確認が必要です。
```

## 9. 開発順序

次の順番で作る。

1. 月別収支 server API
2. 月別収支 一覧画面
3. 月別収支 CSV出力
4. 月別収支 印刷
5. レポートに今月・今年の収支追加
