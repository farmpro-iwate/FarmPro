# Starter Pack 11 バックアップ操作メモ

## 追加されたこと

バックアップ画面を追加しました。

## バックアップできるデータ

- 牛台帳
- 子牛
- 繁殖
- ワクチン
- BLV
- 予定
- 治療

## バックアップ

バックアップ画面で「バックアップJSONをダウンロード」を押します。

## 復元

バックアップ画面で「バックアップJSONを選択して復元」を押し、保存しておいたJSONファイルを選びます。

## 注意

復元すると、現在のJSONデータはバックアップファイルの内容で上書きされます。
復元前に、今のデータもバックアップしてください。

## API

GET http://localhost:4000/api/backups/export

POST http://localhost:4000/api/backups/import
