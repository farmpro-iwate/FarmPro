# FarmPro 本番公開前の安全設定

本番環境では、次の環境変数を必ず設定します。

- `NODE_ENV=production`
- `FARMPRO_AUTH_SECRET`：十分に長いランダム文字列
- `FARMPRO_DATA_DIR`：永続ディスク上のデータ保存先
- `FARMPRO_ALLOWED_ORIGIN`：公開するFarmPro画面のHTTPS URL
- `PORT`：公開サービス側から指定される場合は、その値を使用

## 安全ガード

- 本番で `FARMPRO_AUTH_SECRET` がない場合、サーバーは起動しません。
- 本番で `FARMPRO_DATA_DIR` がない場合、サーバーは起動しません。
- 本番で `FARMPRO_ALLOWED_ORIGIN` がない場合、サーバーは起動しません。
- 本番で利用者ファイルが空の場合、既知のデモアカウントは自動作成されません。
- 許可した画面URL以外からのAPIアクセスはCORSで拒否します。

## 公開前の確認

1. 公開用の試用アカウントを管理コマンドで作成する。
2. `FARMPRO_DATA_DIR` が永続ディスクを指していることを確認する。
3. バックアップJSONを手元にも保存する。
4. 公開URLからログインできることを確認する。
5. 別農場データが表示されないことを確認する。
6. 再起動後も利用者・牛台帳・販売データが残ることを確認する。
