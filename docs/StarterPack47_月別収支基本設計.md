# Starter Pack 47 月別収支 基本設計

## 1. 目的

出荷・販売管理の売上と、経費管理の支出を月ごとに集計し、農場の経営状況を見える化する。

難しい会計処理ではなく、まずは農場で使いやすい「月別収支表」を作る。

## 2. 使うデータ

### 売上データ

出荷・販売管理の `sales.json` を使う。

集計対象は以下。

```text
status = 販売済み
saleDate が入力されている
salePrice が入力されている
```

### 経費データ

経費管理の `expenses.json` を使う。

集計対象は以下。

```text
paymentDate が入力されている
amount が入力されている
```

## 3. 月別収支で表示する項目

| 項目 | 内容 |
|---|---|
| 年月 | 2026-07 など |
| 売上合計 | 販売済みの販売金額合計 |
| 経費合計 | 経費金額の合計 |
| 差引収支 | 売上合計 - 経費合計 |
| 販売頭数 | 販売済み件数 |
| 平均販売金額 | 売上合計 ÷ 販売頭数 |
| 平均販売体重 | 販売体重の平均 |
| 経費件数 | 経費記録数 |
| 飼料費 | 経費区分が飼料費の合計 |
| 診療・医薬品費 | 診療費 + 医薬品費 |
| 繁殖費 | 種付け・繁殖費 |
| その他経費 | 上記以外の経費 |

## 4. 集計ルール

### 売上の月

売上は `saleDate` を基準にする。

例:

```text
saleDate = 2026-07-15
→ 2026年7月の売上
```

### 経費の月

経費は `paymentDate` を基準にする。

例:

```text
paymentDate = 2026-07-03
→ 2026年7月の経費
```

## 5. 差引収支

計算式:

```text
差引収支 = 売上合計 - 経費合計
```

例:

```text
売上合計 650,000円
経費合計 120,000円
差引収支 530,000円
```

## 6. 表示イメージ

```text
年月        売上合計      経費合計      差引収支      販売頭数
2026-07     650,000円    120,000円    530,000円    1頭
2026-08           0円     80,000円    -80,000円    0頭
```
