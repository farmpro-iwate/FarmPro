# FarmPro 入力ミス防止チェックリスト

## 入力前チェック

- 対象の牛・子牛を間違えていない
- 個体番号を確認した
- 日付を確認した
- 入力する画面を間違えていない

## 牛台帳チェック

- 個体番号が半角数字または統一ルールになっている
- 生年月日が正しい
- 父牛・母牛を逆にしていない
- BLV状態が正しい

## 子牛管理チェック

- 子牛番号が重複していない
- 母牛名が正しい
- 性別が正しい
- 出生体重・現在体重が数字になっている
- 生年月日が正しい

## 繁殖管理チェック

- 発情日と授精日を間違えていない
- 種雄牛名の表記がそろっている
- 妊娠結果が正しい
- 分娩予定日が不自然でない

## ワクチン管理チェック

- 対象牛・対象子牛が正しい
- ワクチン名の表記がそろっている
- 接種日が正しい
- 次回予定日が正しい
- 接種済みなら状態を更新した

## BLV管理チェック

- 検査日が正しい
- 検査結果が正しい
- 陽性牛のメモを入力した
- 次回検査予定を入力した

## 予定管理チェック

- 予定区分が正しい
- 予定日が正しい
- 完了した予定は「完了」にした
- 必要ならメモを入力した

## 治療管理チェック

- 治療日が正しい
- 薬剤名がわかるように入力されている
- 投薬量が入力されている
- 休薬終了日が正しい
- 経過が正しい

## GitHub保存前チェック

- アプリが白い画面になっていない
- server と client に赤いエラーが出ていない
- 主要画面が開ける
- 入力したデータが保存されている

問題なければ GitHub に保存します。

```bash
git status
git add .
git commit -m "Update farm data"
git push
```
