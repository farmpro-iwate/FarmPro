# 経費管理 入力ルール

## 1. 基本ルール

経費は、あとで見返したときに内容が分かるように入力する。

特に大事なのは以下。

- 支払日
- 経費区分
- 内容
- 金額
- 支払先

## 2. 支払日

実際に支払った日を入力する。

請求書の日付ではなく、農場で管理したい基準日に統一する。

例:

```text
2026-07-03
```

## 3. 経費区分

なるべく毎回同じ区分を使う。

例:

```text
配合飼料 → 飼料費
ワクチン → 医薬品費
獣医診療 → 診療費
精液代 → 種付け・繁殖費
軽油 → 燃料費
```

## 4. 内容

短く具体的に書く。

良い例:

```text
配合飼料 7月分
子牛用ワクチン
牛舎扇風機修理
人工授精料
```

避けたい例:

```text
支払い
いろいろ
買ったもの
```

## 5. 金額

税込金額で入力する。

例:

```text
120000
```

円マークやカンマは入力しない。

良い例:

```text
120000
```

避けたい例:

```text
￥120,000
120,000円
```

## 6. 支払先

業者名・店名・JAなどを入力する。

例:

```text
JA
〇〇飼料
〇〇動物病院
ホームセンター
```

## 7. 支払方法

以下から選ぶ。

- 現金
- 口座振替
- 銀行振込
- クレジットカード
- JA精算
- その他

## 8. 対象

特定の牛に関係する場合は、個体番号や名前を入力する。

群全体に関係する場合は、群名や用途を入力する。

例:

```text
母牛全体
育成牛群
子牛
1234 はなこ
```

## 9. メモ

補足情報を書く。

例:

```text
7月分
請求書あり
領収書あり
分割払い1回目
```

## 10. 入力後の確認

登録後に以下を確認する。

- 日付が合っているか
- 金額の桁が間違っていないか
- 経費区分が合っているか
- 支払先が分かるか
- 月別収支に反映されそうか
