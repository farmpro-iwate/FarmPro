# FarmPro 個体番号ルール

## 目的

個体番号の表記をそろえるためのルールです。

個体番号がバラバラだと、検索・カルテ・予定・治療記録とのつながりが弱くなります。

## 基本方針

1頭につき、1つの表記にそろえます。

おすすめ:

```text
1234
```

同じ牛で次のように混ぜないようにします。

```text
1234
No.1234
１２３４
1234番
```

## 成牛の個体番号

耳標番号を基本にします。

例:

```text
1234
5678
```

## 子牛の番号

成牛と区別しやすい番号がおすすめです。

例:

```text
C-001
C-002
2026-001
2026-002
```

## 導入牛

導入牛は、導入時の番号をメモに残すと便利です。

例:

```text
個体番号: 7890
メモ: 2026年7月導入
```

## 不明な場合

不明な場合は、仮番号をつけてもOKです。

例:

```text
TEMP-001
仮-001
```

ただし、正式番号がわかったら修正します。

## 番号を変更した場合

メモに旧番号を残します。

例:

```text
旧番号: TEMP-001
正式番号へ変更
```

## 注意

同じ個体を別番号で二重登録しないように注意してください。

二重登録すると、繁殖・治療・ワクチンの記録が分かれてしまいます。
