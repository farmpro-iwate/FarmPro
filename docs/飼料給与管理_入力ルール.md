# 飼料給与管理 入力ルール

## 1. 基本ルール

飼料給与は、後で見返したときに「どの群に、何を、どれだけ与えたか」が分かるように入力する。

特に大事なのは以下。

- 給与日
- 対象
- 飼料名
- 給与量
- 単位

## 2. 給与日

実際に飼料を与えた日を入力する。

例:

```text
2026-07-03
```

## 3. 対象

対象は個体名でも群名でもよい。

例:

```text
母牛群
育成牛群
子牛群
1234 はなこ
```

最初は群単位の入力をおすすめする。

## 4. 飼料名

毎回同じ名前で入力する。

良い例:

```text
配合飼料
乾草
稲わら
牧草ロール
子牛用配合
```

避けたい例:

```text
エサ
いつもの
ロールっぽいもの
```

## 5. 給与量

数字だけで入力する。

良い例:

```text
120
5
```

避けたい例:

```text
120kg
5袋
だいたい少し
```

## 6. 単位

単位はなるべく統一する。

候補:

- kg
- 袋
- ロール
- 束
- 個
- その他

## 7. 単価

単価が分かる場合に入力する。

例:

```text
80
```

金額計算をする場合は、以下の式を使う。

```text
金額 = 給与量 × 単価
```

## 8. 給与目的

以下から選ぶ形にする。

- 維持
- 増体
- 繁殖
- 分娩前
- 子牛育成
- その他

## 9. メモ

補足を書く。

例:

```text
朝夕合計
分娩前なので増量
残飼少しあり
食い込み良好
```

## 10. 入力後の確認

登録後に以下を確認する。

- 日付が合っているか
- 対象が分かるか
- 飼料名が分かるか
- 数量の桁が間違っていないか
- 単位が合っているか
