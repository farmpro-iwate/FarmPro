# 群管理 設計メモ

## 1. 目的

牛を群単位で管理し、飼料給与や作業記録を入力しやすくする。

## 2. 群の例

- 母牛群
- 子牛群
- 育成牛群
- 分娩前群
- 治療中群
- 放牧群

## 3. 登録項目

| 項目 | 内容 |
|---|---|
| 群名 | 母牛群など |
| 区分 | 母牛、子牛、育成牛など |
| 頭数 | 群の頭数 |
| 場所 | 牛舎・放牧地 |
| メモ | 補足 |

## 4. 飼料給与との連携

飼料給与登録で、対象として群を選べるようにする。

例:

```text
対象: 母牛群
飼料名: 配合飼料
給与量: 120kg
```

## 5. 将来的な連携

将来的には、個体台帳や子牛台帳から群へ所属させる。

例:

```text
個体番号 1234 → 母牛群
子牛番号 C001 → 子牛群
```

## 6. 最初の実装方針

最初は簡単に、群名・区分・頭数・場所・メモだけで管理する。

個体との完全な紐づけは後回しにする。
