# Starter Pack 116 種付・受精卵移植 入力項目整理

## 1. 目的

人工授精・自然交配・受精卵移植をFarmProで管理するために、必要な入力項目を整理します。

---

## 2. 共通項目

どの繁殖区分でも必要な項目です。

```text
繁殖記録ID
母牛ID
母牛名
繁殖区分
実施日
分娩予定日
妊娠鑑定予定日
妊娠鑑定結果
状態
メモ
```

---

## 3. 繁殖区分

選択肢:

```text
人工授精
自然交配
受精卵移植
その他
```

---

## 4. 人工授精の項目

```text
授精日
母牛
種雄牛
精液番号
授精師
授精回数
分娩予定日
妊娠鑑定予定日
妊娠鑑定結果
メモ
```

## 4-1. 入力例

```text
繁殖区分: 人工授精
授精日: 2026-07-05
母牛: 1234 はなこ
種雄牛: 福之姫
精液番号: FKH-001
授精師: 〇〇
分娩予定日: 2027-04-16
妊娠鑑定結果: 未鑑定
```

---

## 5. 自然交配の項目

```text
交配日
母牛
種雄牛
同居開始日
同居終了日
分娩予定日
妊娠鑑定予定日
妊娠鑑定結果
メモ
```

## 5-1. 入力例

```text
繁殖区分: 自然交配
交配日: 2026-07-05
母牛: 1234 はなこ
種雄牛: 5678 たろう
同居期間: 2026-07-01 〜 2026-07-10
分娩予定日: 2027-04-16
```

---

## 6. 受精卵移植の項目

受精卵移植では、母牛というより受卵牛の管理が重要です。

```text
移植日
受卵牛
ドナー牛
種雄牛
受精卵番号
新鮮卵 / 凍結卵
受精卵ランク
移植者
同期化処置
黄体確認
分娩予定日
妊娠鑑定予定日
妊娠鑑定結果
メモ
```

## 6-1. 入力例

```text
繁殖区分: 受精卵移植
移植日: 2026-07-05
受卵牛: 1234 はなこ
ドナー牛: 9999 さくら
種雄牛: 福之姫
受精卵番号: ET-2026-001
卵区分: 凍結卵
ランク: A
移植者: 〇〇
分娩予定日: 2027-04-16
妊娠鑑定結果: 未鑑定
```

---

## 7. 妊娠鑑定項目

```text
鑑定予定日
鑑定実施日
鑑定結果
鑑定者
次回確認日
メモ
```

鑑定結果の選択肢:

```text
未鑑定
妊娠
不受胎
再確認
流産
不明
```

---

## 8. 状態管理

繁殖記録の状態は以下がおすすめです。

```text
実施済み
鑑定待ち
妊娠
不受胎
再確認
分娩済み
中止
```

---

## 9. アラート候補

今後作りたいアラート:

```text
妊娠鑑定予定日が近い
妊娠鑑定予定日を過ぎた
分娩予定日が近い
分娩予定日を過ぎた
不受胎後に再種付が未登録
```
