# FarmPro 叩き台完成チェックリスト

FarmPro をいったん叩き台として使える状態にして、実際の動作確認と使用感から次のアップデートを決めるためのチェックリストです。

## 目的

- 細かいコード整形はいったん止める
- 主要機能が一通り開けるか確認する
- 実際の繁殖農家の作業手順で使いやすいか確認する
- 白画面、操作しにくい画面、スマホで使いにくい画面を洗い出す
- 次に直す場所を「使用感」から決める

## 1. 起動確認

- [ ] サーバーを起動できる
- [ ] ブラウザで FarmPro を開ける
- [ ] 白画面にならない
- [ ] 画面上部に「繁殖Farm Pro」が表示される
- [ ] 上部ナビが表示される
