# Starter Pack 114 今後のスマホ改善候補

## 1. 目的

今後、FarmProをスマホでより使いやすくするための改善候補を整理します。

---

## 2. 優先度 高

## 2-1. ホームのスマホ表示改善

改善したいこと:

```text

期限アラートをカード表示にする

```

理由:

```text
ホームは現場で最初に見る画面だから
```

```text

```

```text

```

理由:

```text

```
