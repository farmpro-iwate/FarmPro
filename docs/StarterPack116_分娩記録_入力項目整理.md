# Starter Pack 116 分娩記録 入力項目整理

## 1. 目的

分娩記録をFarmProでしっかり管理するために、必要な項目を整理します。

---

## 2. 分娩記録が必要な理由

分娩記録があると、以下が管理しやすくなります。

```text
母牛の繁殖成績
子牛の出生情報
死産・難産の記録
初乳確認
子牛台帳への登録
次回繁殖計画
```

---

## 3. 基本項目

```text
分娩記録ID
母牛ID
母牛名
関連する繁殖記録ID
分娩予定日
実分娩日
予定日との差
分娩結果
メモ
```

## 10. 子牛台帳への連携

分娩記録から子牛台帳へ登録できるようにしたいです。

連携する項目:

```text
子牛名
耳標番号
性別
生年月日
出生体重
母牛ID
母牛名
父牛
メモ
```

理想の流れ:

```text
分娩記録を登録
↓
子牛情報を入力
↓
子牛台帳にも登録するか確認
↓
子牛台帳へ自動登録
```

## 13. 注意点

分娩記録は、子牛台帳と重複しやすいです。

そのため、今後実装するときは以下に注意します。

```text
同じ子牛を二重登録しない
母牛と子牛を正しくひも付ける
繁殖記録を分娩済みに更新する
死産の場合は子牛台帳に入れるか選べるようにする
```
