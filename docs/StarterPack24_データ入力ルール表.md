# FarmPro データ入力ルール表

## 目的

FarmProに入力するデータの表記をそろえるためのルールです。

表記がバラバラになると、検索・絞り込み・カルテ表示・CSV出力で見つけにくくなります。
できるだけ同じ書き方で入力すると、後から確認しやすくなります。

---

# 1. 牛台帳の入力ルール

## 個体番号

原則として、耳標番号や農場で使っている管理番号を入力します。

おすすめ:

```text
1234
```

避けたい例:

```text
No.1234
１２３４
1234番
```

数字はできるだけ半角でそろえます。

## 牛名

名号や呼び名を入力します。

おすすめ:

```text
さくら
第1花子
```

牛名がない場合は空欄でもOKです。

## 生年月日

日付はカレンダー入力、または次の形式でそろえます。

```text
2026-07-03
```

## 父牛・母牛

わかる範囲で入力します。
不明な場合は空欄でOKです。

## BLV状態

表記はアプリの選択肢に合わせます。

```text
未検査
陰性
陽性
```

---

# 2. 子牛管理の入力ルール

## 子牛番号

成牛の個体番号と混ざらないように、農場内でわかりやすい番号を使います。

おすすめ:

```text
C-001
2026-001
```

## 性別

選択肢に合わせます。

```text
オス
メス
去勢
```

## 体重

kg単位で入力します。
単位「kg」は入力欄には入れず、数字だけにします。

おすすめ:

```text
35
82.5
```

避けたい例:

```text
35kg
約35
```

## DG計算

出生体重と現在体重、生年月日が入っていると、DG確認に役立ちます。

---

# 3. 繁殖管理の入力ルール

## 発情日・授精日

日付は次の形式でそろえます。

```text
2026-07-03
```

## 種雄牛

できるだけ同じ名前で入力します。

おすすめ:

```text
福之姫
安福久
```

避けたい例:

```text
ふくのひめ
福の姫
福之姫号
```

同じ種雄牛なのに表記が違うと、後で集計しにくくなります。

## 妊娠結果

選択肢に合わせます。

```text
未鑑定
妊娠
不受胎
再鑑定
```

## 分娩予定日

授精日から自動計算される場合は、基本的にそのまま使います。
手入力で直す場合は、日付を確認してから変更します。

---

# 4. ワクチン管理の入力ルール

## 対象牛

成牛または子牛の入力補助を使って選びます。

## ワクチン名

表記をそろえます。

おすすめ:

```text
5種混合
IBR
BVD
```

## 接種日・次回予定日

日付形式で入力します。

```text
2026-07-03
```

## 状態

選択肢に合わせます。

```text
予定
接種済み
延期
```

---

# 5. BLV管理の入力ルール

## 検査結果

選択肢に合わせます。

```text
未検査
陰性
陽性
```

## 次回検査予定

次回確認する日付を入れておくと、アラートやカレンダー確認に役立ちます。

## 隔離メモ

陽性牛の管理方針や注意点を短く入力します。

例:

```text
分離飼養
吸血昆虫対策強化
導入牛と接触させない
```

---

# 6. 予定管理の入力ルール

## 予定区分

選択肢に合わせます。

```text
分娩
ワクチン
BLV検査
妊娠鑑定
治療
その他
```

## タイトル

後から見てわかる名前にします。

おすすめ:

```text
妊娠鑑定
5種混合ワクチン
BLV再検査
```

## 状態

完了したら必ず「完了」にします。

```text
未完了
完了
```

---

# 7. 治療管理の入力ルール

## 症状

短く、具体的に入力します。

例:

```text
食欲低下
発熱
跛行
下痢
乳房炎疑い
```

## 診断名

獣医師の診断がある場合に入力します。

## 薬剤

薬剤名はできるだけ正式名称で入力します。

## 投薬量

数字と単位がわかるように入力します。

例:

```text
10ml
2錠
1本
```

## 休薬終了日

出荷・販売に関わる重要項目です。
必ず確認して入力します。

## 経過

選択肢に合わせます。

```text
治療中
経過観察
回復
要再診
```

---

# 8. 共通ルール

## 日付

できるだけこの形式でそろえます。

```text
YYYY-MM-DD
```

例:

```text
2026-07-03
```

## 数字

半角数字でそろえます。

おすすめ:

```text
1234
35.5
```

避けたい例:

```text
１２３４
三十五
約35
```

## 空欄

わからない項目は、無理に適当な値を入れず空欄でOKです。

## メモ

メモは短く、後から見てわかる内容にします。

おすすめ:

```text
導入時から小柄
分娩後注意
次回診療時に確認
```

---

# 9. 入力後の確認

入力したら、次を確認します。

- 個体番号が間違っていないか
- 日付がずれていないか
- 対象牛・対象子牛が正しいか
- 予定日・休薬終了日が正しいか
- 状態が未完了のまま残っていないか

---

# 10. 迷ったとき

迷った場合は、無理に細かく書きすぎず、まず最低限の情報を入れます。

最低限おすすめ:

- 対象牛
- 日付
- 内容
- 状態
- メモ

あとから追記・修正できます。
