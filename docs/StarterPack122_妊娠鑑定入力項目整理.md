# Starter Pack 122 妊娠鑑定 入力項目整理

## 1. 目的

妊娠鑑定で入力する項目を整理します。

---

## 2. 基本項目

```text
繁殖記録ID
母牛ID
母牛名
繁殖区分
実施日
妊娠鑑定予定日
妊娠鑑定実施日
妊娠鑑定結果
鑑定者
次回確認日
状態
メモ
```

---

## 3. 妊娠鑑定結果

選択肢:

```text
未鑑定
妊娠
不受胎
再確認
流産
不明
```

---

## 4. 状態

妊娠鑑定結果に合わせて、繁殖記録の状態も更新します。

```text
鑑定待ち
妊娠
不受胎
再確認
流産
分娩済み
中止
```

---

## 5. 入力例

## 5-1. 妊娠

```text
母牛: 1234 はなこ
繁殖区分: 人工授精
実施日: 2026-07-05
妊娠鑑定予定日: 2026-08-14
妊娠鑑定実施日: 2026-08-15
妊娠鑑定結果: 妊娠
状態: 妊娠
メモ: 鑑定済み。分娩予定管理へ。
```

## 5-2. 不受胎

```text
母牛: 5678 さくら
繁殖区分: 受精卵移植
実施日: 2026-07-10
妊娠鑑定予定日: 2026-08-19
妊娠鑑定実施日: 2026-08-20
妊娠鑑定結果: 不受胎
状態: 不受胎
メモ: 次回発情確認後、再移植検討。
```

## 5-3. 再確認

```text
母牛: 9999 もも
繁殖区分: 自然交配
実施日: 2026-07-01
妊娠鑑定予定日: 2026-08-10
妊娠鑑定実施日: 2026-08-10
妊娠鑑定結果: 再確認
状態: 再確認
次回確認日: 2026-08-20
メモ: 10日後に再確認。
```

---

## 6. 結果別の次の行動

| 鑑定結果 | 次の行動 |
|---|---|
| 未鑑定 | 鑑定予定日を確認 |
| 妊娠 | 分娩予定管理へ |
| 不受胎 | 再種付・再移植を検討 |
| 再確認 | 次回確認日を登録 |
| 流産 | 経過観察・治療記録へ |
| 不明 | メモを残して再確認 |

---

## 7. 画面であると便利なボタン

```text
妊娠にする
不受胎にする
再確認にする
流産にする
メモを追加
既存の繁殖一覧へ戻る
```

---

## 8. 注意点

妊娠鑑定結果を変更すると、分娩予定やアラートにも影響します。

特に以下は注意します。

```text
妊娠 → 分娩予定アラート対象
不受胎 → 分娩予定アラート対象外
再確認 → 再確認アラート対象
流産 → 分娩予定アラート対象外
```

---

## 9. 今後の実装方針

最初は一覧確認画面を作り、その後に結果更新画面を作ります。

```text
Starter Pack 123：妊娠鑑定 一覧確認画面
Starter Pack 124：妊娠鑑定 結果更新画面
```
