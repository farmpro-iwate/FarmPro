# Starter Pack 108 給与目安表 入力項目整理

## 1. 目的

実際の給与目安表をFarmProに入れる前に、どの項目をどこに入れるか整理します。

---

## 2. 現在FarmProにある項目

現在の給与目安で使える項目です。

```text
日齢
ステージ名
スターターkg
育成配合kg
粗飼料kg
メモ
```

---

## 3. 実際の表によくある項目

実際の子牛育成表や給与目安表には、以下のような項目があります。

```text
日齢
月齢
体重
体高
胸囲
スターター
育成配合
粗飼料
ミルク
代用乳
水
注意事項
```

---

## 4. 入力先の対応表

| 実際の表の項目 | FarmProの入力先 | 備考 |
|---|---|---|
| 日齢 | 日齢 | そのまま入力 |
| 月齢 | 日齢 | 30日換算で入れる |
| スターター | スターターkg | 数字だけ |
| 育成配合 | 育成配合kg | 数字だけ |
| 粗飼料 | 粗飼料kg | 数字だけ |
| 注意事項 | メモ | 文章で入力 |
| 体重 | メモ | 現時点ではメモに記録 |
| 体高 | メモ | 現時点ではメモに記録 |
| 胸囲 | メモ | 現時点ではメモに記録 |
| ミルク | メモ | 現時点ではメモに記録 |

---

## 6. メモの書き方例

体重や体高などは、今はメモに入れます。

例:

```text
体重目安: 90kg / 体高: 85cm / 胸囲: 105cm / 離乳準備
```

ミルクがある場合:

```text
代用乳: 4L / スターター慣らし開始
```

注意事項:

```text
水を自由飲水。スターター摂取量を確認。
```

---

## 7. 将来的に追加した方がよい項目

実運用して必要なら、次の項目を画面に追加できます。

```text
体重目安
体高目安
胸囲目安
ミルク量
代用乳量
水
```

ただし最初から増やしすぎると入力が大変になるため、まずはメモで運用しても大丈夫です。

---

## 8. 入力時の注意

```text
単位は入れない
kgは数字だけ
日齢は半角数字
同じ日齢を重複させない
農場で使う呼び方に統一する
```

---

## 9. おすすめ方針

最初は以下の形がおすすめです。

```text
日齢ごとに1行
スターター・育成配合・粗飼料を中心に入力
体重・体高・胸囲はメモ
```

これで給与アラートの判定が使いやすくなります。
