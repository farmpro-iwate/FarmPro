# Starter Pack 114 現場入力ルール

## 1. 目的

現場でスマホやPCから入力する時のルールを整理します。

FarmProは入力が増えるほど便利になりますが、入力ルールがバラバラだと後で見づらくなります。

---

## 2. 現場で入力する主なもの

```text
飼料給与実績
給与アラート対応記録
治療記録
子牛メモ
予定
経費
```

---

## 3. その場で入れるべきもの

できれば作業当日に入力します。

```text
飼料給与実績
給与アラート対応記録
治療記録
次回確認日
```

理由:

```text
あとで思い出して入力すると間違えやすい
日付がずれやすい
対応内容を忘れやすい
```

---

## 4. 後でまとめてもよいもの

```text
経費
売上
月別収支確認
帳票印刷
```

ただし、領収書やメモは残しておきます。

---

## 5. 給与アラート対応記録の入力ルール

## 5-1. 対応日

実際に確認・対応した日を入れます。

```text
2026-07-05
```

## 5-2. アラート種別

なるべく以下から選びます。

```text
不足気味
多め
実績なし
生年月日なし
給与目安なし
その他
```

## 5-3. 対応内容

例:

```text
給与量を増やした
給与量を減らした
給与実績を登録
様子を見る
相談する
確認のみ
```

## 5-4. 状態

```text
未対応
対応中
対応済み
様子見
再確認必要
```

おすすめ:

```text
終わったら対応済み
まだ見るなら様子見
もう一度確認するなら再確認必要
```

## 5-5. 次回確認日

以下の場合は入れるのがおすすめです。

```text
様子見
再確認必要
対応中
```

---

## 6. メモの書き方

短く、後で分かるようにします。

よい例:

```text
スターターを0.5kg増やした。3日後に確認。
粗飼料を少し減らして様子見。
実績入力漏れだったため登録。
食い込み弱い。明日再確認。
```

避ける例:

```text
対応した
OK
あとで見る
いつもの
```

---

## 7. 子牛名のルール

子牛名は統一します。

おすすめ:

```text
耳標番号
耳標番号 + 呼び名
```

例:

```text
1234
1234 はなこ
```

避ける例:

```text
はなこ
花子
ハナコ
1234花子
```

同じ子牛を別名で入力すると、履歴が分かれます。

---

## 8. スマホ入力で気をつけること

```text
変換ミスに注意
数字の全角に注意
日付の選択ミスに注意
保存前に子牛名を見る
保存後に一覧で確認する
```

---

## 9. 毎日続けるためのルール

完璧に入力しようとしすぎると続きません。

最初は以下だけで十分です。

```text
給与実績
気になる子牛の対応記録
次回確認日
対応済みへの更新
```

続けられる形を優先します。
