# FarmPro 主要画面の不足確認

## 目的

FarmProをまず一通り完成させるために、現在のルート構成から主要画面の揃い具合を確認する。

見る観点は以下に絞る。

- 一覧があるか
- 新規登録があるか
- 編集があるか
- 詳細があるか
- 業務の次の画面へつながるか

## すぐに機能追加しない理由

多くの業務で、一覧・新規・編集はすでに揃っている。
そのため、いきなり詳細画面を増やすより、以下を先に確認する。

1. 登録した記録が一覧に出るか
2. 既存記録を編集できるか
3. 牛台帳・子牛カルテで関連記録を確認できるか
4. 今日やることが予定・アラートに出るか
5. CSV・印刷で最低限持ち出せるか

## 次に見るべき穴

### 最優先

1. 子牛カルテに関連記録が集まっているか
2. 牛詳細に関連記録が集まっているか
3. 妊娠確認の登録・編集導線が足りているか

### 次点

4. 予定・カレンダー・アラートの役割整理
5. 月次収支の流れ確認
6. 印刷・CSVの最低限確認

## Starter Pack 220候補

次は以下のどれかを選ぶ。

### 候補A: 子牛カルテの完成度確認

理由:
- 分娩記録から子牛台帳へつながる流れの終点になる
- アプリの価値が見えやすい

見る内容:
- 子牛基本情報
- 母牛情報
- 分娩記録
- 治療記録
- ワクチン記録
- 出荷・販売情報

### 候補B: 牛詳細の完成度確認

理由:
- すべての業務の起点になる
- 母牛単位で繁殖・分娩・治療を追えるかが重要

見る内容:
- 牛基本情報
- 繁殖記録
- 分娩記録
- 治療記録
- ワクチン記録
- 予定

### 候補C: 妊娠確認の導線確認

理由:
- 繁殖から分娩予定につながる重要部分
- ここが曖昧だと分娩記録につながりにくい

見る内容:
- 妊娠確認を登録できるか
- 編集できるか
- 繁殖記録とつながっているか
- 分娩予定へつながるか

## 結論

現時点で、画面数はかなり揃っている。
次からは新しい画面を増やすより、詳細画面やカルテに関連記録が集まっているかを確認する。

次のおすすめは、**子牛カルテの完成度確認**。
