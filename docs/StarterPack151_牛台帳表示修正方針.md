# Starter Pack 151 牛台帳表示修正方針

## 1. 目的

牛台帳の表示を、耳標番号・個体識別番号中心に整理します。

---

## 2. 基本方針

画面で主に見る番号:

```text
耳標番号
```

正式な管理番号:

```text
個体識別番号
```

画面にできるだけ出さない言葉:

```text
ID
```

---

## 3. 牛台帳で表示したい項目

理想は以下です。

```text
耳標番号
個体識別番号
名前
生年月日
性別
状態
産次数
BLV状態
メモ
```

ただし、最初から全部は追加しません。

まずは表示名の整理を優先します。

---

## 4. 既存データとの対応

今までの内部データ名は、しばらくそのまま使います。

```text
id → 内部用
earTag → 耳標番号
name → 名前
cattleId → 内部用または旧管理番号
cowId → 内部用または母牛耳標番号
```

---

## 5. 画面で直したい言葉

```text
牛ID
母牛ID
個体ID
ID
個体番号
```

これらは、画面ではなるべく使いません。

---

## 6. 画面で使いたい言葉

```text
耳標番号
母牛耳標番号
個体識別番号
```

---

## 7. 次の実装候補

次のPackでは、確認結果を見てから以下のどれかを直します。

```text
牛一覧画面
牛登録画面
牛編集画面
個体カルテ
```

一番安全なのは、まず牛一覧画面の表示名だけ直すことです。

---

## 8. 注意

内部の項目名を変えるのは後回しです。

まず画面表示だけ直します。

これなら、既存データを壊しにくいです。
