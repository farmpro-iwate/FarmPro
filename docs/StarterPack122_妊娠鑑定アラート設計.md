# Starter Pack 122 妊娠鑑定アラート設計

## 1. 目的

妊娠鑑定の見落としを減らすため、アラート設計を整理します。

---

## 2. 作りたいアラート

```text
妊娠鑑定予定日が近い
妊娠鑑定予定日が今日
妊娠鑑定予定日を過ぎた
再確認日が近い
再確認日を過ぎた
不受胎後の再種付未登録
```

---

## 3. 優先度

## 3-1. 高


## 3-2. 中


## 3-3. 低


---

## 4. アラート判定の例

## 4-1. 鑑定予定日超過

条件:

```text
pregnancyResult = 未鑑定
pregnancyCheckDate < 今日
```

表示:

```text
妊娠鑑定予定日を過ぎています
```

## 4-2. 今日鑑定

条件:

```text
pregnancyResult = 未鑑定
pregnancyCheckDate = 今日
```

表示:

```text
今日は妊娠鑑定予定日です
```

## 4-3. 再確認超過

条件:

```text
pregnancyResult = 再確認
nextCheckDate < 今日
```

表示:

```text
再確認予定日を過ぎています
```
