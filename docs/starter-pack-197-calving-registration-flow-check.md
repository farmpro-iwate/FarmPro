# Starter Pack 197: 分娩記録から子牛カルテまでの導線確認

分娩記録一覧は一旦完成扱いにし、ここからは実際の登録導線を確認する。

## 確認する流れ

1. `/calvings/new` で分娩記録を新規登録する
2. 保存後、通常の分娩は `/calvings?registration=ready` に移動する
3. 登録候補に保存した記録が表示される
4. `登録してカルテへ進む` を押す
5. 確認ダイアログで母牛、母牛耳標番号、子牛耳標番号、性別、出生日、出生体重を確認する
6. OK後、上部に `子牛カルテを開く` が表示される
7. 子牛カルテを開き、耳標番号、母牛、生年月日、開始体重を確認する
8. 子牛カルテに分娩記録由来の表示が出るか確認する

## OK条件

- 保存後に迷わず登録候補へ行ける
- 登録候補でどの記録を子牛台帳へ登録するか分かる
- 登録前に内容確認ができる
- 登録後に一覧から探し直さず子牛カルテへ行ける
- 子牛カルテで分娩記録から登録された子牛だと分かる

## 重点確認

- 子牛耳標番号が画面上で一貫して見えること
- 母牛ID、子牛IDなどの内部IDが主表示になっていないこと
- 死産は登録候補に出ず、登録対象外として扱われること
- 要確認の記録は登録前に不足項目が分かること

## 次に直す候補

実動作確認で詰まった場所を優先する。

- 登録後の子牛カルテリンクが分かりにくい
- 登録候補に出る条件が分かりにくい
- 子牛カルテで由来表示が出ない
- 入力必須項目が厳しすぎる、または足りない
- スマホでボタンが押しにくい
