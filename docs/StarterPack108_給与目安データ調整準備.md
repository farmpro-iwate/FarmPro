# Starter Pack 108 給与目安データ調整準備

## 1. 目的

FarmProには、飼料給与目安表の機能があります。

現在は、日齢ごとに以下を管理できます。

```text
日齢
ステージ名
スターター目安
育成配合目安
粗飼料目安
メモ
```

ただし、現在の給与目安には仮データが含まれている可能性があります。

そのため、実際の農場で使う前に、農場の資料・飼料会社の資料・実際の給与方針に合わせて調整する必要があります。

---

## 2. いきなり上書きしない理由

給与目安は、以下に影響します。

```text
ホームの給与アラート
全子牛給与アラート
子牛カルテの給与目安
レポートの給与アラート集計
対応記録作成
```

そのため、数値を間違えて入れると、アラート判定もずれてしまいます。

まずは以下の順番で進めます。

```text
1. 現在のデータを確認
2. 実際の表の項目を整理
3. 変更前バックアップ
4. 置き換える数値を決める
5. データ更新
6. 画面で確認
```

---

## 3. 現在の給与目安データ確認

Chromeで開きます。

```text
http://localhost:5173/feeding-guide
```

確認するもの:

```text
日齢
ステージ名
スターター
育成配合
粗飼料
メモ
```

server側のデータファイルはこちらです。

```text
server/src/data/feedingGuide.json
```

---

## 4. 実際の表で確認する項目

実際の給与目安表から、以下を確認します。

```text
日齢または月齢
体重目安
体高目安
胸囲目安
スターター給与量
育成配合給与量
粗飼料給与量
ミルクまたは代用乳
注意事項
```

FarmProに今すぐ入れられる項目:

```text
日齢
ステージ名
スターター
育成配合
粗飼料
メモ
```

今後追加した方がよい項目:

```text
体重目安
体高目安
胸囲目安
ミルク
代用乳
```

---

## 5. データ調整方針

最初は、複雑にしすぎず以下を優先します。

```text
日齢
ステージ名
スターター
育成配合
粗飼料
メモ
```

体重・体高・胸囲などは、次の段階で追加してもよいです。

理由:

```text
まず給与アラート判定に使っている項目を安定させるため
```

---

## 8. 注意点

## 8-1. kgの入力

画面には数値だけ入れます。

よい例:

```text
0.5
1.2
2
```

避ける例:

```text
0.5kg
1.2キロ
少し
```

---

## 8-2. 日齢の重複

同じ日齢を何度も登録しないようにします。

例:

```text
90日
90日
90日
```

このように重複すると、近い目安を選ぶ時に分かりにくくなります。

---

## 8-3. 実績との差

現在のアラート判定は、目安より大きくずれた場合に不足・多めとして見ます。

そのため、目安が厳しすぎるとアラートが多くなります。

逆に、目安が大ざっぱすぎるとアラートが出にくくなります。
