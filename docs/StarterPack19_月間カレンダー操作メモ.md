# Starter Pack 19 月間カレンダー 操作メモ

## 追加されたこと

月間カレンダーを追加しました。

## 表示される予定

- 予定管理の予定日
- 繁殖記録の分娩予定日
- ワクチンの次回予定日
- BLVの次回検査予定日

## 使い方

1. 上部メニューの「カレンダー」を押す
2. 月間カレンダーが表示される
3. 「前月」「翌月」で月を切り替える
4. 「印刷する」でカレンダーを印刷できる

## ホーム画面

ホーム画面の作業ショートカットにも「カレンダー」を追加しました。

## メモ

予定は、日付が入力されているものだけ表示されます。
接種済みワクチンはカレンダーに表示しません。
