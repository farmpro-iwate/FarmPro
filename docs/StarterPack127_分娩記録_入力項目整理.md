# Starter Pack 127 分娩記録 入力項目整理

## 1. 目的

分娩記録で入力する項目を整理します。

---

## 2. 最初に使う項目

最初は以下だけに絞ります。

```text
母牛
分娩予定日
実分娩日
子牛名
性別
出生体重
分娩結果
初乳確認
メモ
```

---

## 3. 入力項目の詳細

### 母牛

分娩した母牛を選びます。

```text
母牛ID
母牛名
```

既存の牛台帳から選べると便利です。

### 分娩予定日

種付・受精卵移植から計算された予定日です。

既存の繁殖記録から引き継げると便利です。

### 実分娩日

実際に分娩した日です。

### 予定日との差

これは自動計算でよいです。

```text
実分娩日 - 分娩予定日
```

例:

```text
-2日
0日
+3日
```

### 子牛名

子牛台帳へつなげるための名前です。

```text
子牛名
耳標番号
```

最初は1つの入力欄でもOKです。

### 性別

選択肢:

```text
オス
メス
不明
```

### 出生体重

数字だけ入力します。

よい例:

```text
32
```

避ける例:

```text
32kg
32キロ
```

単位は画面側で `kg` と表示します。

### 分娩結果

選択肢:

```text
正常
要確認
死産
中止
```

### 初乳確認

選択肢:

```text
未確認
確認済み
要確認
```

### メモ

細かい内容はメモに書きます。

例:

```text
自然分娩。初乳確認済み。
少し弱いので明日確認。
介助あり。母牛は良好。
死産。母牛は経過観察。
```

---

## 4. 最初は入れない項目

以下は後回しです。

```text
後産確認
獣医師
介助者
難産スコア
治療内容
体温
詳細な母牛状態
詳細な子牛状態
```

---

## 5. 入力例

```text
母牛: 1234 はなこ
分娩予定日: 2027-04-16
実分娩日: 2027-04-15
予定日との差: -1日
子牛名: 1234-1
性別: メス
出生体重: 32
分娩結果: 正常
初乳確認: 確認済み
メモ: 自然分娩。初乳確認済み。
```

---

## 6. 実運用での使い方

分娩直後に全部入れようとしなくてもよいです。

まず入れるもの:

```text
母牛
実分娩日
子牛名
性別
分娩結果
```

あとで入れてもよいもの:

```text
出生体重
初乳確認
メモ
```

---

## 7. 結論

分娩記録は、正確さも大事ですが、続けられることがもっと大事です。

最初は少ない項目で始めます。
