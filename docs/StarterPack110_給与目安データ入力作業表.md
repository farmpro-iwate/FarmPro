# Starter Pack 110 給与目安データ入力作業表

## 1. 目的

実際の給与目安表をFarmProに入れる前に、ここで数値を整理します。

---

## 2. 入力対象

FarmProで使う主な項目です。

```text
日齢
ステージ名
スターターkg
育成配合kg
粗飼料kg
メモ
```

---

## 4. メモに入れるとよい内容

実際の表に以下がある場合は、まずメモに入れます。

```text
体重目安
体高目安
胸囲目安
代用乳
ミルク量
注意事項
```

例:

```text
体重目安: 90kg / 体高: 85cm / 胸囲: 105cm / 離乳後の食い込み確認
```

---

## 5. 月齢から日齢にする目安

```text
1か月 = 30日
2か月 = 60日
3か月 = 90日
4か月 = 120日
5か月 = 150日
6か月 = 180日
```

完全に正確な日数でなくても、給与目安としては区切りが分かりやすいことが大切です。

---

## 6. 数値入力の注意

```text
数字は半角
単位は入れない
kgは書かない
空欄にしすぎない
同じ日齢を重複させない
```

---

## 7. 入力後に確認すること

```text
スターター量が急に大きく変わりすぎていない
育成配合の開始時期が表と合っている
粗飼料の開始時期が表と合っている
離乳前後の数値が自然につながっている
```
