# 販売記録の入力ルール

## 目的

出荷・販売管理を追加したときに、入力をそろえるためのルールです。

---

# 1. 対象区分

対象区分は次のようにそろえます。

```text
子牛
成牛
その他
```

## 子牛

子牛市場や販売に出した子牛です。

## 成牛

母牛、繁殖牛、肥育牛などの成牛です。

## その他

上記に当てはまらない場合に使います。

---

# 2. 対象番号

子牛の場合は子牛番号、成牛の場合は個体番号を入力します。

おすすめ:

```text
C-001
1234
2026-001
```

避けたい例:

```text
No.1234
１２３４
1234番
```

番号は、牛台帳・子牛管理と同じ表記にします。

---

# 3. 販売日

日付は次の形式にそろえます。

```text
YYYY-MM-DD
```

例:

```text
2026-07-03
```

---

# 4. 出荷日

販売日と同じ日なら同じ日付を入れます。

出荷だけ先に行う場合は、出荷日と販売日を分けます。

---

# 5. 販売先

販売先は表記をそろえます。

例:

```text
〇〇家畜市場
〇〇牧場
〇〇農協
```

同じ販売先を、毎回違う書き方にしないようにします。

---

# 6. 市場名

市場を通した場合に入力します。

例:

```text
岩手県南家畜市場
全農市場
```

市場を通していない場合は空欄でもOKです。

---

# 7. 販売体重

kg単位で数字だけ入力します。

おすすめ:

```text
285
312.5
```

避けたい例:

```text
285kg
約300
三百
```

---

# 8. 販売金額

円単位で数字だけ入力します。

おすすめ:

```text
650000
720000
```

避けたい例:

```text
65万円
720,000円
約70万
```

カンマや円は入れず、数字だけにすると集計しやすいです。

---

# 9. 購買者

わかる範囲で入力します。

個人名や会社名を入力する場合は、農場内の管理ルールに従います。

---

# 10. 状態

状態は次のようにそろえます。

```text
出荷予定
出荷済み
販売済み
取消
```

## 出荷予定

まだ出荷していない状態です。

## 出荷済み

出荷したが、販売結果がまだ未入力の状態です。

## 販売済み

販売金額や体重まで入力済みの状態です。

## 取消

出荷予定がなくなった場合です。

---

# 11. メモ

必要に応じて短く入力します。

例:

```text
市場出荷
体重良好
出荷前確認済み
休薬問題なし
```

---

# 12. 入力後チェック

入力後に確認します。

- 対象番号が正しい
- 販売日が正しい
- 販売体重が数字
- 販売金額が数字
- 状態が正しい
- 休薬期間中ではない
- 治療中ではない
