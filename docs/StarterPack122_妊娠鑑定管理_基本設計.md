# Starter Pack 122 妊娠鑑定管理 基本設計

## 1. 目的

繁殖管理では、種付や受精卵移植を登録するだけでなく、その後の妊娠鑑定がとても重要です。

FarmProでは、今後以下を管理できるようにします。

```text
妊娠鑑定予定日
妊娠鑑定実施日
鑑定結果
再確認日
不受胎後の対応
流産・中止
分娩予定へのつながり
```

---

## 2. 妊娠鑑定管理が必要な理由

妊娠鑑定をきちんと管理すると、以下が分かりやすくなります。

```text
鑑定待ちの母牛
鑑定日を過ぎた母牛
妊娠した母牛
不受胎だった母牛
再確認が必要な母牛
分娩予定に進む母牛
再種付が必要な母牛
```

---

## 3. 基本の流れ

```text
種付・受精卵移植を登録
↓
妊娠鑑定予定日を自動計算
↓
鑑定予定日が近づいたらアラート
↓
鑑定結果を入力
↓
結果に応じて状態を更新
```

---

## 4. 鑑定結果ごとの動き

## 4-1. 未鑑定

まだ鑑定していない状態です。

```text
状態: 鑑定待ち
アラート対象: はい
```

## 4-2. 妊娠

妊娠確認済みです。

```text
状態: 妊娠
分娩予定管理へ進む
アラート対象: 分娩予定アラートへ移行
```

## 4-3. 不受胎

妊娠していない状態です。

```text
状態: 不受胎
再種付検討
分娩予定から外す
```

## 4-4. 再確認

もう一度鑑定が必要な状態です。

```text
状態: 再確認
次回確認日を入力
再確認アラート対象
```

## 4-5. 流産

妊娠していたが途中で流産した状態です。

```text
状態: 流産
分娩予定から外す
治療・経過観察が必要な場合あり
```

---

## 5. 妊娠鑑定予定日の自動計算

目安として、実施日から40日後を妊娠鑑定予定日にします。

```text
妊娠鑑定予定日 = 実施日 + 40日
```

ただし、農場の方針や獣医師の判断により、手動で変更できるようにします。

---

## 6. 管理したい一覧

今後作ると便利な一覧:

```text
鑑定待ち一覧
今日鑑定予定一覧
鑑定予定日超過一覧
再確認一覧
不受胎一覧
妊娠確認済み一覧
```

---

## 7. 既存画面を壊さない方針

妊娠鑑定管理も、最初は別画面で作ります。

候補URL:

```text
/pregnancy-checks
```

または、繁殖強化の中で扱う場合:

```text
/breedings-advanced/pregnancy-checks
```

既存の以下は触りません。

```text
/breedings
/breedings/new
/breedings/:id/edit
```

---

## 8. 次に作るもの

次は、妊娠鑑定の一覧確認画面を作るのがおすすめです。

```text
Starter Pack 123：妊娠鑑定 一覧確認画面
```

表示したいもの:

```text
母牛名
繁殖区分
実施日
妊娠鑑定予定日
予定日までの日数
鑑定結果
状態
分娩予定日
```

---

## 9. 結論

妊娠鑑定を整理すると、繁殖管理がかなり実用的になります。

次からは、鑑定待ち・期限切れ・再確認を見やすくする方向で進めます。
