# Starter Pack 52 飼養管理 基本設計

## 1. 目的

繁殖和牛農家向けに、日々の飼料給与・飼料在庫・群管理を記録し、飼料費や飼養状況を見える化する。

まずは難しい栄養計算ではなく、農場で実際に使いやすい「飼養記録」として作る。

## 2. 飼養管理でできるようにすること

将来的に以下をアプリへ追加する。

- 飼料給与一覧
- 飼料給与新規登録
- 飼料給与編集
- 飼料給与削除
- 飼料名で検索
- 対象群で絞り込み
- 飼料在庫管理
- 群管理
- 月別飼料使用量集計
- 月別飼料費集計
- CSV出力
- 印刷
- レポート反映

## 3. 飼料給与管理の登録項目

| 項目 | 内容 | 例 |
|---|---|---|
| 給与日 | 飼料を与えた日 | 2026-07-03 |
| 対象 | 個体・子牛・群 | 母牛群 |
| 飼料名 | 飼料の名前 | 配合飼料 |

| 単位 | kg、袋、ロールなど | kg |

| メモ | 補足 | 朝夕合計 |
| 作成日時 | 自動記録 | |
| 更新日時 | 自動記録 | |

## 6. 経費管理との連携

飼料を購入した時点では、経費管理に「飼料費」として登録する。

飼料給与管理では、実際に使った量を記録する。

将来的には以下を連携する。

```text
経費管理：飼料を買った金額
飼料在庫：入庫した数量
飼料給与：使った数量
月別収支：飼料費として集計
```

## 7. 月別飼料費へのつなげ方

飼料給与記録から、月ごとに以下を集計する。

- 飼料使用量
- 飼料費
- 対象群別飼料費
- 飼料名別使用量
- 飼料名別費用

## 8. レポート画面への追加案

レポート画面には、以下を追加する。

- 今月飼料費
- 今年飼料費
- 飼料使用量
- 在庫不足候補
- 対象群別飼料費

## 9. CSV出力

飼養管理では以下をCSV出力できるようにする。

- 飼料給与CSV
- 飼料在庫CSV
- 群管理CSV
- 月別飼料費CSV

## 10. 印刷

以下を印刷できるようにする。

- 飼料給与台帳
- 飼料在庫表
- 群管理表
- 月別飼料費表

## 11. 開発順序

安全に進めるため、次の順番で作る。

1. 飼養管理 server API
2. 飼料給与一覧画面
3. 飼料給与新規登録
4. 飼料給与編集
5. 飼料給与削除
6. 検索・絞り込み
7. CSV出力
8. 印刷
9. 飼料在庫管理
10. 群管理
11. レポート反映
12. 月別飼料費集計

## 12. 注意

最初は記録しやすさを優先する。

栄養計算や飼料設計は将来的な拡張とし、まずは「何を、いつ、どれだけ、いくら使ったか」を記録する。
