# FarmPro Starter Pack 45 経費管理 印刷追加版

## 内容

Starter Pack 45では、経費管理に印刷機能を追加します。

## 更新ファイル

- client/src/pages/ExpenseList.tsx
- docs/StarterPack45_経費管理印刷_確認手順.md
- README_StarterPack45_ExpensePrint.md

## 追加されること

- 経費一覧に「印刷」ボタンを追加
- 現在表示中の検索・絞り込み結果を印刷
- 経費台帳として紙やPDFに保存
- 印刷日時を表示
- 表示件数・経費合計を表示
- 印刷時は編集・削除などの操作ボタンを非表示
