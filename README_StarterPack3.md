# 繁殖Farm Pro - Starter Pack 3 子牛管理

Starter Pack 3 では、子牛管理を追加します。

## 追加内容
- 子牛一覧
- 子牛新規登録
- 子牛編集
- 子牛削除
- 生年月日から日齢を自動計算
- 開始体重・現在体重・経過日数からDG（日増体量）を自動計算
- ミルク量
- スターター給与量
- JSON保存

## 反映方法
このZIPの中身を、既存の FarmPro フォルダへ上書きコピーしてください。

## 起動方法
serverフォルダ:
```bash
npm install
npm run dev
```

clientフォルダ:
```bash
npm install
npm run dev
```

## GitHub保存
```bash
git add .
git commit -m "Add Starter Pack 3 calf management"
git push
```
