# 繁殖Farm Pro - Starter Pack 10 レポート・CSV出力

Starter Pack 10 では、各データを集計して確認できるレポート画面を追加します。

## 追加内容

- レポートメニュー
- 経営・作業サマリー
- 登録牛数
- 子牛数
- 繁殖記録数
- ワクチン記録数
- BLV陽性頭数
- 未完了予定数
- 治療中頭数
- 休薬中件数
- 近日予定
- CSV出力
  - 牛台帳CSV
  - 子牛CSV
  - 繁殖CSV
  - ワクチンCSV
  - BLV CSV
  - 予定CSV
  - 治療CSV

## 反映先

C:\Users\関口\Downloads\FarmPro_StarterPack1\FarmPro_StarterPack1

server:
