# 繁殖Farm Pro - Starter Pack 8 予定管理

Starter Pack 8 では、農場の予定管理を追加します。

## 追加内容

- 予定管理メニュー
- 予定一覧
- 予定の新規登録
- 予定の編集
- 予定の削除
- 予定区分
  - 分娩
  - ワクチン
  - BLV検査
  - 妊娠鑑定
  - 治療
  - その他
- 予定日
- 対象牛
- 完了 / 未完了
- 期限判定
  - 期限超過
  - 今日
  - まもなく
  - 予定あり
  - 完了
- JSON保存
