# 繁殖Farm Pro - Starter Pack 16 子牛選択・入力補助

Starter Pack 16 では、登録済みの子牛を選んで入力できる補助機能を追加します。

## 追加内容

- 子牛選択コンポーネント
- 登録済み子牛から選択
- 子牛番号・子牛名を自動入力
- 手入力も継続可能
- 対応画面
  - ワクチン登録
  - 予定登録
  - 治療登録

## 便利になること

Starter Pack 15 では成牛の選択補助を追加しました。
Starter Pack 16 では子牛も選択できるようにします。

ワクチン・予定・治療で、成牛だけでなく子牛も選択しやすくなります。
